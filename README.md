# Expandly 1 (Previously TextExpander v1)
This was the initial version of the project, able to do the bare minimum, without looking pretty or being easy to work with.

# History
TextExpander v1 started out with just expansions and triggers, no variables, and hardcoded expansions (meaning you would need to enter the python script to make edits).
Upon the release of v1.1.0 (to my knowledge), variables were introduced, alongside a JSON based data saving format for variables, expansions and triggers.

# Do not install this version.
This version has been made public **for historical reasons only** and is not intended for actual use outside of seeing the original thoughts of this project.
